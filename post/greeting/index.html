Список основных команд git

Вот список основных команд Git с описанием:

  1. git init
    Инициализация нового репозитория Git в текущей папке. Создаёт скрытую папку .git для управления версией.
    Пример:

    git init

  2. git clone <URL>
    Клонирование удалённого репозитория на локальный компьютер.
    Пример:

    git clone https://github.com/username/repo.git

  3. git status
    Показывает статус текущего репозитория: изменённые файлы, файлы, готовые к коммиту, и файлы, не отслеживаемые Git.
    Пример:

    git status

  4. git add <файл>
    Добавляет файл в индекс (стейджинг). Этот файл будет включён в следующий коммит.
    Пример:

    git add file.txt

    Для добавления всех файлов:

    git add .

  5. git commit -m "<сообщение>"
    Фиксирует изменения в репозитории с сообщением.
    Пример:

    git commit -m "Добавил новую функцию"

  6. git log
    Показывает историю коммитов.
    Пример:

    git log

  7. git push
    Отправляет изменения (коммиты) в удалённый репозиторий.
    Пример:

    git push origin main

  8. git pull
    Загружает последние изменения из удалённого репозитория и сливает их с текущей веткой.
    Пример:

    git pull origin main

  9. git branch
    Управление ветками: создание, удаление и просмотр списка веток.
    Пример:

    • Показать текущие ветки: 
      git branch
    • Создать новую ветку: 
      git branch new-branch

  10. git checkout <ветка>
    Переключение между ветками.
    Пример:

    git checkout main

  11. git merge <ветка>
    Сливает указанную ветку с текущей.
    Пример:

    git merge new-branch

  12. git remote -v
    Показывает список удалённых репозиториев (например, GitHub).
    Пример:

    git remote -v

  13. git fetch
    Загружает изменения из удалённого репозитория, но не сливает их с локальной веткой.
    Пример:

    git fetch origin

  14. git diff
    Показывает разницу между текущими изменениями и последним коммитом.
    Пример:

    git diff

  15. git reset
    Сбрасывает изменения (удаляет их из индекса или отменяет коммит).
    Пример:

    git reset file.txt

  16. git rm <файл>
    Удаляет файл из репозитория и из файловой системы.
    Пример:

    git rm file.txt

  17. git stash
    Временно сохраняет изменения, не готовые к коммиту, и очищает рабочую директорию.
    Пример:

    git stash

  18. git config
    Настройка параметров Git (например, имя пользователя и email).
    Пример:

    git config --global user.name "Ваше имя"
git config --global user.email "email@example.com"
