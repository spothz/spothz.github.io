Перенос порта ssh с 22.

Чтобы изменить порт для SSH с 22 на 44333, выполните следующие шаги:

1. Измените конфигурацию SSH

Откройте конфигурационный файл SSH-сервера для редактирования:

sudo nano /etc/ssh/sshd_config

2. Измените строку с портом

Найдите строку, которая выглядит как:

#Port 22

Удалите комментарий (если есть) и измените номер порта на 44333:

Port 44333

3. Сохраните изменения и выйдите

Нажмите Ctrl + X, затем Y и Enter, чтобы сохранить изменения и выйти из редактора.

4. Разрешите новый порт в брандмауэре

Если у вас включен брандмауэр (например, UFW или firewall), разрешите новый порт:

Для UFW:

sudo ufw allow 44333/tcp

Для других брандмауэров (например, iptables):

sudo iptables -A INPUT -p tcp --dport 44333 -j ACCEPT

5. Перезапустите SSH-сервис

Теперь перезапустите SSH-сервис для применения изменений:

sudo systemctl restart sshd

6. Подключитесь к серверу

Теперь вы можете подключаться к серверу по новому порту с использованием параметра -p:

ssh user@your-server-ip -p 44333

7. (Опционально) Закрытие старого порта (22)

Если вы уверены, что новый порт работает, можно заблокировать порт 22 для дополнительной безопасности. Для этого выполните команду:

sudo ufw deny 22/tcp

или для iptables:

sudo iptables -A INPUT -p tcp --dport 22 -j REJECT

Теперь ваш SSH-сервер будет работать на порту 44333.
