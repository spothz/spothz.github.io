Установка и настройка 3x-ui.

Предполагается, что VPS уже куплен. Если нет - рекомендую брать тут:

  • inferno.name
  • pq.hosting

  1. Обновляем репозитории и софт на сервере:

     apt update && apt upgrade -y

  2. Устанавливаем панель 3x-ui

     bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)

В процессе установки выбираем y в ответ на вопрос хотим ли внести изменения. Вводим логин, пароль, порт любой в диапазоне 41000-65000, например 63421 и сохраняем полученные данные для авторизации в панели.

  1. Запускаем скрипт, который установит SSL-сертификат в панель для безопасного входа

     bash <(curl -Ls https://raw.githubusercontent.com/cyb3rm4gus/3x-ui-auto_add_ssl/main/3x-ui-autossl.sh)

  2. Установка Cloudflare WARP Вводим команду x-ui, в появившемся меню выбираем пункт 21, затем опцию 1. Порт устанавливаем 40000

  3. Установка fail2ban Вводим команду

    x-ui

для вызова меню, выбираем пункт 20, затем опцию 1, дожидаемся завершения. Перезагрузка панели x-ui выбираем пункт 13

Поздравляю, на этом настройка панели через консоль завершена

Заходим на сервер по адресу: https://ip_сервера:порт/web-base-path, порт и web-base-path вы указывали выше, и (надеюсь) вы их сохранили. Если нет, вызовите меню командой x-ui и выберите пункт 10 для просмотра конфигурации.

После ввода логина и пароля перейдите в раздел настройки xray, выберите WARP и следуйте инструкции из видео.

При указании маршрутизации добавьте:

geosite:category-gov-ru,regexp:.*\.ru$,regexp:.*\.su$`

После этого обязательно нажмите вверху “сохранить” и “перезагрузить”.

Создание клиентов

  • В настройках безопасности выбираем REALITY и жмем Get New Cert ниже.
  • Раскрываем поле “клиент” и в Flow выбираем xtls-rprx-vision
  • Изменяем yahoo.com и www.yahoo.com на google.com и www.google.com
  • В пункте uTLS выбираем chrome
  • После этого жмем создать, конфиг и сервер готовы к использованию.

Устанавливаем фаервол

sudo apt install ufw -y

Разрешаем SSH

sudo ufw allow 22/tcp

Разрешаем панель 3x-ui

sudo ufw allow 2053/tcp

Разрешаем порт для VLESS/XTLS Reality

sudo ufw allow 443/tcp

Включаем UFW

sudo ufw enable

Проверяем статус UFW

sudo ufw status

Для добавления любого порта соотвественно:

ufw allow номер_порта

Скрипт установки официального warp-cli в режиме локального прокси для серверов на Debian и Ubuntu.

➖ Подходит, чтобы настроить варп для 3X-UI или Marzban ➖ После установки на вашем сервере появляется локальный socks прокси на адресе 127.0.0.1:30000 ➖ Установка:

curl -L https://raw.githubusercontent.com/Skrepysh/tools/refs/heads/main/install-warp-cli.sh > install-warp-cli.sh && sudo chmod +x install-warp-cli.sh && sudo ./install-warp-cli.sh

Чтобы добавить warp-cli в 3X-UI добавьте outbound (исходящий) со следующими параметрами: https://raw.githubusercontent.com/spothz/Wiki/refs/heads/main/IMG_20250228_102003_384.jpg?token=GHSAT0AAAAAACOOAV656NA36XI5MLHP3QYIZ6BCQBQ
